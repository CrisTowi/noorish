Architecture & Public Deployment Notes
Persistence
10 state keys persisted via usePersistedState hook wrapping localStorage. Keys prefixed noorish_*. Survives page refresh. Cleared on plan quit (week badges only).
Error Boundary
ErrorBoundary class wraps AppInner. Catches runtime errors, shows branded recovery screen. "Refresh app" calls window.location.reload().
Hooks Rule
All hooks declared before early return in AppInner. Onboarding gate (if !onboarded return <Onboarding/>) runs after all useState and useEffect calls.
AI Feature
Claude Sonnet via Anthropic API. Requires meal logs or swaps to activate. Falls back to mock suggestions in preview. Returns structured JSON: add / reduce / swap categories.
Accessibility
All icon-only buttons have aria-label. Minimum font-size 11px throughout. Textarea fontSize:16px prevents iOS zoom. Colour contrast meets WCAG AA.
Before Launch
Remove /* DEMO SHORTCUT */ button from TodayScreen. Verify Anthropic API auth in target environment. Consider adding a Terms / Privacy notice for public users.
