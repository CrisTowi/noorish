Figma Setup Summary 1. Create a local Variables collection Nourish/Tokens with all color tokens above.
2. Create Text Styles: Nourish/Display, Nourish/SheetTitle, Nourish/CardDay, Nourish/Body, Nourish/Meta, Nourish/MacroValue, Nourish/Eyebrow, Nourish/Caption.
3. Create Effect Styles: Nourish/Nav/BlurBg = background blur 20px.
4. Frame size: 390 × 844 (iPhone 14) or 430 × 932 (iPhone 14 Plus). Set frame background to --surface.
5. Icon component: 24×24 frame, outline SVG path, currentColor stroke so it inherits from parent.
